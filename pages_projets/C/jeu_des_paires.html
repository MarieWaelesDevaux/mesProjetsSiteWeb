Durant ma première année de BUT Informatique, notre principale SAE (Situation d'Apprentissage et d'Évaluation) du premier semestre consistait à réaliser un jeu de paires en utilisant la bibliothèque ncurses sur Linux. Cette bibliothèque permet de créer des interfaces graphiques en utilisant uniquement des caractères ASCII dans le terminal.

Pour ce projet, nous devions d'abord implémenter le jeu avec une page d'accueil offrant aux utilisateurs la possibilité de choisir s'ils souhaitaient jouer ou lancer un mode automatique. Nous n'avions pas de consignes particulières pour le mode automatique et pouvions donc le concevoir de manière entièrement aléatoire, ce que j'ai fait dans un premier temps. Cependant, j'ai par la suite opté pour une version plus élaborée, intégrant une stratégie réfléchie et utilisant la mémoire.

Ensuite, nous devions également concevoir une page web pour exposer notre projet. Voir le projet web
