Durant ma première année de BUT Informatique, notre principale SAE (Situation d'Apprentissage et d'Évaluation) du premier semestre consistait à réaliser un jeu de paires en utilisant la bibliothèque ncurses sur Linux. Cette bibliothèque permet de créer des interfaces graphiques en utilisant uniquement des caractères ASCII dans le terminal.

Pour ce projet, nous devions d'abord implémenter le jeu. Voir la page de description de l'implementation du jeu.

Ensuite, nous devions également concevoir une page web (uniquement en HTML et CSS) pour exposer notre projet. Cette page devait expliquer les règles, les instructions d'utilisation, ainsi que nos choix d'implémentation, les difficultés rencontrées, notre organisation, ....
