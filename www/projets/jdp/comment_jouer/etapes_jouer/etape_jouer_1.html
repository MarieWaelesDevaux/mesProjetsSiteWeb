Comment accéder au jeu ?

Avant de commencer, il faut savoir que le jeu fonctionne que sur Linux à cause de la librairie ncurses utilisé pour l'affichage. Néaumoins, il existe une équivalence sous Windows qui adapte le code, la librairie pdcurses.

Il vous est également possible d'installer sous windows, une machine virtuelle Linux en suivant le PDF.

Nous vous avons mis à disposition les fichiers compressé pour Windows et Linux (il vous suffit de cliquer sur un des boutons "Installer"). Voici les étapes à faire pour lancer le jeu :

  • Téléchargez le fichier zip correspondant à votre système d'exploitation.
  • Décompressez le.
  • Dans votre terminal, placez vous dans le dossier décompressé (à l'aide de la commande cd).
  • Assurez vous que la taille de votre fenêtre du terminal soit assez grande.
  • Entrez les commandes suivantes :
    • Sous windows : 
      
 .\jdp.exe
    • Sous Linux : 
      
 chmod +x jdp
 ./jdp
